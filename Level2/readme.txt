Please follow these steps:

Unzip to desktop
- Update content in index.html to be specific to your web environment
- Search and replace "Your website" with the actual name of your website
- Update the brand zone and navigation. 
- Alternatively, add your header and footer as server side include files


DEVELOPMENT SERVER
- Upload entire contents to DEV
- Upload the robots.txt file to root
- We encourage you to apply a team wsso policy to your entire dev site

PRODUCTION SERVER
- Edit redirect.html with an absolute link to your DEV /_admin directory
- Rename redirect.html to index.html
- Upload the /_admin/index.html

Analytics

There are multiple analytics applications in use across the enterprise.

  • Weblog Expert is the standard for EWH websites.
  • WebTrends common on Communications servers
  • Coremetrics is used externally
  • Google Analytics is replacing Coremetrics externally and on high visibility internal websites
  • Adobe Analytics

If using Weblog Expert, please be aware that it has several issues right out of the box. It's best to request some modifications via a MyServices request.

  • 90 days of logfile processing (vs the standard 30 days).
  • Exclusion filters edited to disallow crawlers and robots.
  • Create a CSV of the data every 3 months.

System for Tracking Automated Requests (STAR)

The STAR system allows you to create or change ownership of Global Groups. A Global Group is for managing membership for permissions to access web shares for developing and file management. Global Groups do not manage access from a browser to the website itself - that is a function of WSSO.

Start with the System for Tracking Automated Requests (STAR) website.

  • Choose BAAM Windows
    • In the left side menu: ‘Initiate request for Yourself’. [Next >]
  • In the scrolling selection box choose: ‘Global/Univ (GGM) Group – CREATE’. This will populate the box on the right. [Next >]
  • Click on Global or Universal group request.xlsx to get the excel spreadsheet.
  • Complete the spreadsheet. See attached example. In this example Steve, Diane and I along with a business partner are identified as both owners and managers)
    • Select the action to ‘Create – New Global Group’. Do NOT choose ‘Universal’ group if hosting on EWH – it cannot be implemented in that environment.
    • Always include our three core domains MW, NW, SW (there's also EU, AU, and NE domains)
  • If necessary, include any other domains of business partners who need access to the web share (NE, SE, …other
    • Name the Group.
      • I like to use a naming convention which relates to the website name: i.e., boeingcollaborationcenter for boeingcollaborationcenter.web.boeing.com
      • Or a name that relates to the membership, i.e.,: AquentCSdev or cswebstl_webauth
    • Once all relevant domains are identified add the owners and managers. Identify multiple owners – typically peers or key business partners. That way there is no single point of failure in the event someone is unable to support
      • Owners can request changes to the status of the group – whether or not to add domains or delete the group entirely or pass ownership to someone else. Owners must also be identified as managers if they need to administer the group membership
      • Managers can add members to the group.
  • Save the excel file. I like to use a naming convention which relates to the website name: boeingcollaborationcenter_ggm_create.xlsx
  • Submit the STAR Request. You will receive an email stating whether or not the request is accepted or rejected. Sometimes the request may be rejected because someone has already chosen the Group name you wanted.
  • If accepted, you will receive another email stating that the request is completed. (typically within a day or sooner)
